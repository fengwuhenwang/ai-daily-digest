## [V4.2.1-full] - 2026-02-20

### 重大升级
- **FinanceManagerAgent 完整专业版本**
  - 完整目录结构（agents/finance_manager/）
  - 专业化配置体系
  - 数据驱动闭环系统

### 新增文件结构
```
finance_manager/
├── agent.yaml              # Agent配置文件
├── prompt.md               # Agent人格定义
├── tools.yaml              # 工具配置（Tiingo + Alpha Vantage）
├── scheduler.yaml          # 调度器配置
├── workflow.yaml           # 工作流配置
├── scoring.py              # 三维评分算法（Python）
├── memory.json             # 记忆模板
├── data/
│   ├── portfolio.json      # 投资组合数据
│   ├── history_nav.json    # 历史净值
│   └── market_cache.json   # 市场数据缓存
└── reports/
    └── weekly/
        └── weekly_report_template.md  # 周报模板
```

### 核心功能
- **三维评分系统**（加权综合评分）
  - 回撤维度（40%权重）
  - 趋势维度（30%权重）
  - 波动维度（20%权重）
  - 相关性维度（10%权重）

- **机会雷达**（智能触发）
  - 30日跌幅 ≥ 10%
  - 7日跌幅 ≥ 5%
  - 距60日高点回撤 ≥ 15%
  - 三维评分 ≥ 70

- **情景分支资产配置**
  - 乐观情景（VOO 65% / BND 30% / 黄金 5%）
  - 中性情景（VOO 60% / BND 30% / 黄金 10%）
  - 悲观情景（VOO 55% / BND 35% / 黄金 10%）

- **汇率调整收益系统**
  - USD/CNY汇率追踪
  - 真实人民币收益计算

- **长期绩效追踪**
  - 年化收益率（CAGR）
  - 最大回撤
  - 夏普比率
  - 波动率

- **自动再平衡系统**
  - 偏离阈值：20%
  - 平台建议：IB、Tiger、雪球

- **风险监控系统**
  - 标普500单周跌幅 > 5%
  - VIX指数上升
  - 跌破200日均线

### 监控资产
- VOO, QQQ, BND, IAU, VNQ（美股）
- KWEB, FXI（中国）
- SOXX, SMH（半导体）
- USD/CNY（汇率）
