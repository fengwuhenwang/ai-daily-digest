# Meta Learning Engine

## 概述
分析用户决策模式，生成认知画像。

## 更新触发
- 每月1次（自动）
- 手动触发：`meta update`

## 分析维度

### 1. 决策质量趋势 (Decision Quality Trend)
- 计算：最近N次决策的平均得分
- 趋势：上升/下降/稳定

### 2. 置信度-准确度差距 (Confidence Accuracy Gap)
- 计算：(自我评估准确率 - 实际准确率) × 100
- 正值：过度自信 | 负值：自信不足 | 0：校准良好

### 3. 风险准确度 (Risk Accuracy Score)
- 计算：预测风险与实际风险的匹配度
- 范围：0-100

### 4. 决策速度 (Decision Velocity)
- 平均决策时间
- 速度评级：过快/适中/过慢

### 5. 环境分析
- 优势环境：成功率>70%的决策类型/场景
- 劣势环境：成功率<40%的决策类型/场景

### 6. Meta Learning Score
- 综合评分：0-100
- 组件：自我觉察、学习速度、偏差修正

## 隐私原则
- 不暴露内部机制
- 只输出可执行的洞察
- Token预算：单次分析<500 tokens
