# Goal Calibration Engine

## 概述
建立显性目标函数系统，确保系统始终知道自己在优化什么。

## 核心功能

### 1. 长期目标定义
- **能力跃迁** (35%): 认知深度、决策质量、执行效率
- **财富跃迁** (35%): 资产增长、收入多元化、财务安全
- **人生自主度** (30%): 时间自由、选择权、关系质量

### 2. Utility权重参数化
四大维度加权：
- capability (50%)
- wealth (45%)
- autonomy (30%)
- risk_penalty (负权重)

### 3. Non-Negotiables
不可牺牲的底线：
1. 健康安全
2. 财务安全
3. 法律合规
4. 心理稳定

### 4. Alignment Score
- 计算：当前行为与目标函数的匹配度
- 阈值：
  - >0.6: 正常
  - 0.4-0.6: 警告
  - <0.4: 触发ASE重构

### 5. 校准频率
- 每季度自动校准
- 手动触发：`goal recalibrate`

## 隐私原则
- 不暴露内部推理机制
- 输出只包含可执行洞察
- Token预算：单次分析<500 tokens

## 与其他引擎关系
- ASE: Alignment下降时触发战略重构
- DSE: 决策质量评估输入
- Cognitive Profile: 学习用户如何优化目标
