# Decision Weight Engine

## 动态权重模型

基于人生阶段、风险敞口、资产规模自动调整评分权重。

### 权重触发条件

| 触发器 | 阈值 | 动作 |
|--------|------|------|
| 人生阶段变化 | - | 重新加载阶段权重 |
| 风险敞口变化 | ±20% | 调整风险识别度权重 |
| 资产规模变化 | ±30% | 调整执行路径权重 |

### 人生阶段权重配置

- **探索期**: 重视信息完整度 + 执行路径
- **积累期**: 重视风险识别 + 目标清晰度
- **突破期**: 重视目标清晰度 + 执行路径
- **跃迁期**: 重视目标清晰度 + 信息完整度
- **稳定期**: 均衡权重

### DSE 集成

每次评分前必须：
1. 读取 `memory/decision_weight_model.json`
2. 获取当前 `current_weight`
3. 使用动态权重计算最终得分
4. 记录权重调整历史

### 计算公式

```
weighted_score = Σ(dimension_score × current_weight / base_weight)
```

### 权重调整记录

每次调整必须记录：
- 调整时间
- 触发原因
- 调整前后权重对比
- 决策者确认
